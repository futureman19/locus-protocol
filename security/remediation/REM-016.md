# REM-016: Propagate RNG Errors

**Fixes:** FINDING-016
**Priority:** LOW
**Effort:** Small (< 15 minutes)

## Current Code

```rust
pub fn secure_random(&self) -> [u8; 32] {
    let mut buf = [0u8; 32];
    getrandom::getrandom(&mut buf).unwrap_or_default();
    buf
}
```

## Recommended Fix

```rust
pub fn secure_random(&self) -> Result<[u8; 32], GhostError> {
    let mut buf = [0u8; 32];
    getrandom::getrandom(&mut buf)
        .map_err(|e| GhostError::runtime(format!("RNG failure: {}", e)))?;
    Ok(buf)
}
```

The error will propagate to the ghost, which can handle it appropriately (e.g., refuse to perform cryptographic operations without randomness).
